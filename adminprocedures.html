Admin Emergency Protocols

Code Blue - Normal Protocol
Indicates regular functioning. All activities permitted.

Download Scanner: OFF

Download logger remains on

Server User Scan: 1m

Code Number: 1

Code Green - Server Scanning Protocol
Heightened alert for suspicious activity. Normal activities permitted.

Download Scanner: ON

Download logger remains on

Server User Scan: 30s

Code Number: 2

Code Yellow - Unauthorized Access Attempt
Indicates an attempt to breach server access.

Download Scanner: ON

Download logger remains on

Outside Firewall: ON until Code Yellow or below

Main Logs: Normal To Debug. Change back when yellow or below

Server User Scan: 15s

Code Number: 3

Code Red - Intrusion Detected
Confirmation of unauthorized presence within the servers.

Download Scanner: ON

Download logger remains on

Outside Firewall: ON until Code Yellow or below

Inside Firewall: ON until Code Red or below

Main Logs: Normal/Debug To Error. Change back when red or below

Server User Scan: 5s

Code Number: 4

Code Black - Complete Takeover
Servers compromised, total loss of control.

Download Scanner: ON

Download logger remains on

Outside Firewall: ON until Code Yellow or below

Inside Firewall: ON until Code Red or below

Bunker Firewall: ON

Main Logs: Normal/Debug/Error To Fatal Error. Change back when red or below

Announce Code Black

Declare an Emergency

Server User Scan: 1s

Code Number: 5
